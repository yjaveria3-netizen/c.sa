# C Language Examples

This directory contains C language examples written without built-in functions, demonstrating fundamental programming concepts and implementations from scratch.

## 📁 Directory Structure

### 🔍 [algorithms/](./algorithms/)
- **binary search** - Binary search algorithm implementation
- **bubble sort** - Bubble sort algorithm
- **fibonacci** - Fibonacci sequence generator
- **prime sieve** - Sieve of Eratosthenes for prime numbers

### 🏗️ [data_structures/](./data_structures/)
- **linked list** - Linked list implementation from scratch

### 📝 [string_operations/](./string_operations/)
- **palindrome** - Palindrome checker without string.h
- **reverse string** - String reversal function
- **string length** - Manual string length calculation

### 🔢 [mathematical/](./mathematical/)
- **matrix multiplication** - Matrix multiplication algorithm
- **power function** - Power function without math.h

### 🎮 [games/](./games/)
- **guess game** - Number guessing game

### 📚 [examples/](./examples/)
- **c** - Basic C language example

## 🎯 Learning Objectives

These examples teach:
- **Memory management** - Manual allocation and deallocation
- **Pointer manipulation** - Direct memory access
- **Algorithm implementation** - From scratch without library functions
- **System programming** - Low-level concepts
- **Performance optimization** - Efficient code writing

## 💡 Key Features

- **No built-in functions** - All implementations are from scratch
- **Memory-safe** - Proper memory management demonstrated
- **Educational focus** - Clear, well-commented code
- **Progressive difficulty** - From basic to advanced concepts

## 🚀 Getting Started

1. Start with **examples/c** for basic syntax
2. Move to **string_operations** for fundamental operations
3. Progress to **algorithms** for complex problem-solving
4. Explore **data_structures** for advanced concepts

## 📖 Usage

Each `.sa` file can be run with the Quantum Language interpreter:
```bash
quantum example.sa
```

## 🎓 Recommended Learning Path

1. **Beginner**: examples → string_operations → mathematical
2. **Intermediate**: algorithms → data_structures
3. **Advanced**: games (combining all concepts)
