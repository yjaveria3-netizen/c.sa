# C Library Examples

This directory contains comprehensive C-style implementations of common algorithms, data structures, and programming examples written in Quantum Language.

## Directory Structure

### 📁 algorithms/
Classic computer science algorithms and problem-solving techniques.

#### Files:
- **binary search.sa** - Binary search implementation with variations
- **bubble sort.sa** - Bubble sort algorithm with optimizations
- **fibonacci.sa** - Fibonacci sequence calculations (recursive, iterative, memoized)
- **heap sort.sa** - Heap sort algorithm implementation
- **insertion sort.sa** - Insertion sort with stability analysis
- **linear search.sa** - Linear search with multiple variants
- **merge sort.sa** - Merge sort algorithm with divide-and-conquer
- **prime sieve.sa** - Sieve of Eratosthenes for prime generation
- **quick sort.sa** - Quick sort with partitioning strategies
- **selection sort.sa** - Selection sort algorithm
- **depth first search.sa** - DFS traversal for graphs and trees
- **breadth first search.sa** - BFS traversal for graphs and trees

### 📁 data_structures/
Fundamental data structures and their operations.

#### Files:
- **linked list.sa** - Singly linked list with CRUD operations
- **stack.sa** - Stack implementation with applications
- **queue.sa** - Queue implementation with variants
- **binary tree.sa** - Binary tree and BST with traversals
- **hash table.sa** - Hash table with collision handling

### 📁 examples/
Practical programming examples and real-world applications.

#### Files:
- **c.sa** - Basic C syntax demonstration
- **file operations.sa** - File I/O operations and utilities
- **memory management.sa** - Memory allocation and management simulation

### 📁 games/
Game implementations and algorithmic game theory.

#### Files:
- **guess game.sa** - Number guessing game with hints
- **tic tac toe.sa** - Tic Tac Toe with AI opponent
- **snake game.sa** - Snake game with collision detection
- **rock paper scissors.sa** - Rock Paper Scissors with strategy

### 📁 mathematical/
Mathematical algorithms and numerical computations.

#### Files:
- **matrix multiplication.sa** - Matrix operations and multiplication
- **power function.sa** - Power function implementations
- **factorial.sa** - Factorial calculations with multiple approaches
- **gcd lcm.sa** - Greatest Common Divisor and Least Common Multiple
- **prime numbers.sa** - Prime number generation and testing

### 📁 string_operations/
String manipulation, validation, and processing algorithms.

#### Files:
- **palindrome.sa** - Palindrome detection and generation
- **reverse string.sa** - String reversal algorithms
- **string length.sa** - String length calculation methods
- **string validation.sa** - Input validation for various formats
- **text processing.sa** - Text analysis and transformation
- **text encryption.sa** - Classical encryption algorithms

## Usage

Each file can be run independently with the Quantum Language interpreter:

```bash
quantum examples/library/C/algorithms/binary_search.sa
```

## Learning Path

### Beginner
1. Start with **examples/c.sa** for basic syntax
2. Try **string_operations/string_length.sa** for basic operations
3. Explore **algorithms/linear_search.sa** for simple algorithms

### Intermediate
1. Study **data_structures/stack.sa** and **data_structures/queue.sa**
2. Practice with **algorithms/bubble_sort.sa** and **algorithms/insertion_sort.sa**
